# traitbac


- bacterial traits database and query
- carry entire bacdive and bactotraits on your laptop for parallel threaded search

```
cargo build
```

Usage: traitbac <COMMAND>

Commands:
bacto-i-ds        search pattern bacdive
bacto-ncbi        search ncbi
bacto-ncb-ilevel  search ncbi associated level
bactor-dna        search rrdna
bactotax          search taxonomy
bacto-phylo       search phylum
bacto-class       search class
bacto-order       search order
bacto-family      search family
bacto-genus       search genus
bacto-species     search species
bactorest         search rest
help              Print this message or the help of the given subcommand(s)

Options:
-h, --help     Print help
-V, --version  Print version
